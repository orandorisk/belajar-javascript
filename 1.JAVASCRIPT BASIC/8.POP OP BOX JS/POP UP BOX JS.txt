..ALERT [alert('hello world');] SAAT DIJALAN KAN PADA BROWSER AKAN MUNCUL THIS PAGE SAYS HELLO WORLD DAN BUTTON OK. HALAMAN AKAN BERJALAN SAAT OK DI KLIK
//BUAT INDEX.HTML DAN SCRIPT.JS PADA SCRIPT.JS INPUTKAN [alert('hello world');] SECARA UX POP UP SANGAT NEGATIF EFEK

..PROMPT MEMUNCULKAN WINDOWS KECIL BUKAN HNYA TOMBOL OK NAMUN JUGA INPUTAN [prompt('masukan angka');] SAAT DIJALANKAN AKAN MUNCUL TULISAN MASUKAN NAMA. UNTUK MELIHAT INTERAKSI PADA INPUT SAAT DI OK MUNCUL BUAT SYNTAX
var nama = prompt('masukan nama:');
alert(nama);

..CONFIRM UNTUK MEMINTA CONFIRMASI USER DENGAN PERINTAH [confirm('kamu yakin');] PADA BROWSER HANYA OK DAN CANCEL . FUNGSI CONFIRM MENGEMBALIKAN NILAI BOOLEAN SAAT TEKAN OK TRUE / SAAT CANCEL FALSE . KALAU PROMPT TADI MENGEMBALIKAN APAPUN YG DIINPUT . MEMBUAT SEBUAH SYNTAX SEDERHANA TENTANG TRUE DAN FALSE.

PERCABANGAN
var tes = confirm('apa kamu yakin?');
if (tes === true){
alert('user menekan tombol ok');}
else{alert('user menekan tombol cancel');}
